# 📚 Library Management System (C++)

## 🔹 Features
- Add Book
- View Books
- Search Book
- Delete Book
- Issue Book
- Return Book
- File Storage (Data saved permanently)

## 🔹 Technologies Used
- C++
- File Handling
- STL (Vector)

## 🔹 How to Run
1. Compile:
   g++ main.cpp -o library
2. Run:
   ./library
